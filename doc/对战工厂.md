# 对战工厂

## 游戏玩法
1.玩家入场时给出6只随机配置的宝可梦，任选4只，随机的宝可梦配置尽可能合理（参考一下vgc和网上各种配置）
2.选择完毕后开始7轮对战，每轮胜利后可以选择对手的一只宝可梦和自己的一只宝可梦进行交换
3.每次胜利后将胜利的玩家的队伍存入对手池，作为异步匹配用
4.每轮对战结束后，将对手池中的对手进行匹配
5.如果对手池没有精灵，就随机生成4只配置尽可能合理的宝可梦
6.进行段位区分，段位是精灵球，超级球，高级球，大师球，将对手池也用段位区分开来
7.段位积分暂时设置为2000，达到两千分升段，然后清空当前段位积分，如果低于0分则降低段位
8.每轮对战结束后，将当前段位积分存入数据库
9.初始积分都为0，初始段位是精灵球，达到大师球后开始计算排名，按积分排序，积分相同则按照达到积分的时间排序
